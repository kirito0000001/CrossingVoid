安卓注意事项
旧手机一律下载X64，新的手机，确保自己是Arm64构架的可以下Arm版本
阿里云盘禁止传播Apk和zip文件，所以只能下载PC端

如果出现安装包解析失败可以尝试以下几种方法

1，使用MT管理器进行安装，在应用商店就能下载到
2，下载第二个安装包进行安装（不一定解决全部机型，据测试X86安卓还是安装不了（平板））
3，关闭安装界面的网络检测（因为零境是没有上架的非正规应用，所以网络检测是大概率会被BAN的）
